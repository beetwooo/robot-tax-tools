AI 엔지니어 코스

단계 1: 기초 준비

Python, 수학, 데이터 전처리 기초를 학습합니다.

1

Python과 기초 수학

Python 문법과 선형대수, 통계 기초를 학습하세요.

2

데이터 전처리

결측치 처리, 스케일링, 특징 선택을 연습합니다.

3

기본 머신러닝

회귀, 분류 모델을 구현하고 성능을 평가합니다.

단계 2: 딥러닝 실습

신경망을 구현하고 모델을 개선하는 과정을 학습합니다.

1

신경망 기본

퍼셉트론과 역전파, 손실 함수의 역할을 이해하세요.

2

TensorFlow/Keras

모델을 정의하고 학습시키는 실습을 진행합니다.

3

이미지/텍스트 모델

간단한 이미지 분류와 텍스트 분석 모델을 만들어 보세요.

완성한 모델을 간단한 서비스로 배포합니다.

1

API 만들기

2

웹 연동

간단한 웹 인터페이스로 모델을 사용하는 방법을 구현합니다.

3

서비스 개선

사용자 피드백을 반영하여 모델과 서비스를 개선하세요.
