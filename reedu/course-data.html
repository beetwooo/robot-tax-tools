홈 직무 코스 학습 자료

데이터 분석 코스

데이터 분석가가 되기 위한 단계별 과정과 실전 과제를 제공합니다.

단계 1: 기초 학습

SQL, 엑셀, 파이썬, 통계 기초를 학습하세요.

1

SQL 기초

데이터베이스와 SQL 쿼리를 통해 데이터를 조회하고 분석하는 방법을 배우세요.

2

엑셀과 데이터 정리

엑셀 함수와 피벗 테이블을 활용해 데이터를 정리하고 요약합니다.

3

파이썬 기초

파이썬 문법과 pandas 라이브러리를 사용한 데이터 전처리를 연습하세요.

단계 2: 실습 프로젝트

실제 데이터를 분석하며 결과를 도출하고 보고서를 작성합니다.

1

데이터 수집

공개 데이터셋을 선택하고 데이터를 가져옵니다.

2

데이터 분석

필터링, 집계, 시각화를 통해 데이터를 이해합니다.

3

보고서 작성

분석 결과를 정리하여 발표 자료나 문서로 만듭니다.

단계 3: 포트폴리오

완성한 프로젝트를 정리하여 이력서와 링크드인에 공유하세요.

1

프로젝트 설명

프로젝트 목표와 과정, 사용 기술을 명확히 작성합니다.

2

결과 공유

시각화 자료와 주요 통찰을 보기 쉽게 정리하세요.

3

포트폴리오 등록

Github, Notion, 블로그 등에 프로젝트를 게시합니다.
